# Global Port Public Tariff Search

A bilingual (Chinese/English) port tariff search tool covering **40+ ports** across 7 major shipping regions:

- China (Mainland)
- Taiwan
- Hong Kong
- Southeast Asia
- Red Sea
- Turkey
- Black Sea

**Live URL:** https://Leylalyu-cul.github.io/port-tariff-search/port-tariff-search.html

## Features

- Bilingual search (Chinese / English)
- Filter by region, fee type, currency
- Real-time keyword search
- Paginated results
